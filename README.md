# ReParcel

React Parcel starter template, ready to ship with [Netlify](https://netlify.com/) and [Now](https://now.sh)!

## How to

### Installing dependencies

With yarn:

```bash
yarn
```

Or with npm:

```bash
npm i -s
```

### Running the code

With Yarn:

```bash
yarn start
```

Or with npm:

```bash
npm start
```
